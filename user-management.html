Add New User

Create a brand new user and set their role
1

Step 1: User Identity

Enter a unique username and a corporate email address for the new team member.

2

Step 2: Assigning the Role

This is critical. Choose the correct access level:

  • Editor: Can manage and publish posts/pages.
  • Shop Manager: Can manage products and orders.
  • Administrator: Full access (use sparingly).
Action
3

Step 3: Save User

Click the button to create the account. An invitation email will be sent automatically.

Quick Tip

To change a role later, go to Users > All Users, click the user, and update the 'Role' dropdown.
